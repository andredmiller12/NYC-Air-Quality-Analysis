# NYC-Air-Quality-Analysis

This project analyzes air pollution data from NYC Open Data using Python and Pandas. I analyzed air pollution data to understand potential health impacts and environmental inequality in NYC across boroughs and neighborhoods

Air pollution exposure has been linked to respiratory diseases, cardiovascular diseases, cancer, and premature death. Understanding how air pollution varies across neighborhoods and over time can help identify communities that may be disproportionately affected by environmental risks. This project analyzes NYC Open Data air quality data to examine pollution trends across boroughs, neighborhoods, seasons, and years.

Tools Used:
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

Analysis Performed:
- Data cleaning and transformation
- Pollution trends over time
- Pollution by borough
- Pollution by pollutant type
- Seasonal pollution comparison
- Identification of most polluted neighborhoods

Key Findings:
- Manhattan had the highest nitrogen dioxide levels
- Air pollution decreased over time
- Winter pollution levels were higher than summer levels
- Certain neighborhoods consistently showed higher pollution levels
